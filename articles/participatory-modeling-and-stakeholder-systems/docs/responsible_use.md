# Responsible Use

Participatory modeling can influence public policy, environmental management, infrastructure investment, organizational change, health systems, climate adaptation, and resource governance.

Do not use these synthetic workflows for real decisions without:

- Real stakeholder mapping
- Inclusion of affected groups
- Accessibility and language planning
- Facilitation design
- Consent and data governance
- Documentation of dissent
- Assumption review
- Power and representation analysis
- Clear decision authority
- Shared interpretation of results
- Accountability for model use

Participation should not be used to legitimize decisions that stakeholders cannot actually influence.
