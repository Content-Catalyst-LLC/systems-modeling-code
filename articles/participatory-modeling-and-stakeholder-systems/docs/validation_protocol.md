# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and transparent scoring behavior.

## Checks

- Stakeholder weights are exported.
- Scenario performance table is exported.
- Assumption register is exported.
- Participation-level taxonomy is exported.
- Facilitation and evidence-governance tables are exported.
- Stakeholder weights sum to one.
- Scenario scores remain between zero and one.
- Scenario summary is generated.
- Assumption register is generated.

## Professional extension

Applied participatory modeling should add facilitation logs, attendance records, dissent registers, missing stakeholder analysis, scenario provenance, qualitative interpretation, model walkthroughs, power analysis, data rights, and public accountability mechanisms.
