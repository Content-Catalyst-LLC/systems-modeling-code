# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Stress scenario inventory is generated.
- Strategy stress-test runs are generated.
- Service ratios remain between zero and one.
- Failure shares remain between zero and one.
- Resilience scores remain between zero and one hundred.
- Regret is nonnegative.
- Cumulative unmet demand is nonnegative.
- Recovery time is nonnegative.
- Robustness summary is exported.

## Professional extension

Applied use should add empirical validation, structural review, domain stress evidence, sensitivity analysis, threshold justification, stakeholder review, and uncertainty communication tests.
