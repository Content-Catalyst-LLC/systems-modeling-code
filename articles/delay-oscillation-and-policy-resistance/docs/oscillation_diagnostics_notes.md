# Oscillation Diagnostics Notes

Oscillation is repeated movement around a target, equilibrium, desired stock level, or constraint.

## Useful diagnostics

- Target crossings
- Maximum overshoot above target
- Maximum undershoot below target
- Mean absolute target gap
- Final target gap
- Peak-to-trough range
- Cumulative intervention
- Cumulative counterresponse
- Resistance ratio

Target crossings alone are not enough. A system can oscillate weakly around a target or swing destructively across a wide range.
