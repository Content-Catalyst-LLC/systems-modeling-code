# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Urban component taxonomy is exported.
- Feedback-loop taxonomy is exported.
- Modeling approaches and equity dimensions are exported.
- Scenario trajectories are generated.
- Summary diagnostics are generated.
- Population, housing, transport, service capacity, accessibility, congestion, housing gap, and service pressure remain nonnegative.
- Validation checks are exported.

## Professional extension

Applied use should add local administrative data, transportation data, housing-market data, infrastructure condition data, geospatial layers, historical validation, sensitivity analysis, equity review, and participatory interpretation.
