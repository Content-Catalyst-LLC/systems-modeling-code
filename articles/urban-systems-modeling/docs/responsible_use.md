# Responsible Use

Urban systems models can influence zoning, transit, housing, infrastructure, policing, climate adaptation, public services, and neighborhood investment. They should be used with transparency, humility, and public accountability.

Do not use these synthetic workflows for real decisions without:

- Local data
- Spatial validation
- Calibration against observed trends
- Community participation
- Equity assessment
- Displacement risk review
- Privacy and data governance
- Sensitivity analysis
- Clear communication of limitations

Urban systems modeling should make tradeoffs, assumptions, and consequences visible. It should not hide political choices behind technical language.
