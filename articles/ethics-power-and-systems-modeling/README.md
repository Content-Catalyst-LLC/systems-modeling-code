# Ethics, Power, and Systems Modeling

Advanced companion code for the article **Ethics, Power, and Systems Modeling**.

This folder demonstrates stakeholder coverage diagnostics, power-burden scoring, representation-gap review, governance registers, ethical model-use risk scoring, boundary-power questions, model safeguards, misuse patterns, validation checks, SQL review tables, and multi-language companion workflows.

## Quick start

```bash
./run_all.sh
```

The Python workflow uses only the standard library. The R workflow uses base R. Other language examples run when their compilers/interpreters are installed.

## Interpretation warning

All data are synthetic. These workflows are review scaffolds, not substitutes for real stakeholder engagement, legal review, public accountability, data governance, ethical review, or model validation.
