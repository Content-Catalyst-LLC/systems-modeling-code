# Integrated Assessment Modeling Protocol

Integrated assessment modeling connects human and Earth systems across long time horizons. A transparent workflow should define the assessment question, system boundary, scenario assumptions, architecture, data, uncertainty, ethics, and communication strategy.

Core questions:

- What climate, energy, land, sustainability, or policy question is being examined?
- Which regions, sectors, technologies, land systems, damages, and policy levers are included?
- Which variables are endogenous and which are exogenous assumptions?
- Which scenario assumptions drive the results?
- Which feedback loops are active or omitted?
- Which ethical assumptions are embedded in discounting, damages, welfare, and aggregation?
- How are uncertainty and sensitivity tested?
- How are model limits communicated?
