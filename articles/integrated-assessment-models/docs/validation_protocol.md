# Validation Protocol

The synthetic validation protocol checks that trajectories are generated, summaries are exported, mitigation rates remain between 0 and 1, and output, emissions, atmospheric pressure, temperature proxy, damages, mitigation cost, and consumption remain nonnegative.

Professional IAM validation should add calibration, historical hindcasting where appropriate, scenario protocol documentation, uncertainty ranges, model intercomparison, and sensitivity analysis for discounting, damages, technology learning, deployment constraints, and climate response.
