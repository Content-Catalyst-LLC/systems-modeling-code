# Assumptions and Limitations

All data are synthetic. Output, emissions intensity, mitigation, atmospheric pressure, temperature response, damages, cost, consumption, and welfare are simplified. The climate module is a transparent proxy, not a physical climate model. The damage function is a stylized quadratic proxy. The welfare proxy is illustrative only.

This is not a calibrated integrated assessment model. It does not reproduce DICE, GCAM, IMAGE, MESSAGEix, REMIND, AIM, or any official model. Applied IAM work requires formal model documentation, empirical calibration, validation, sensitivity analysis, uncertainty analysis, model comparison, and ethical review.
