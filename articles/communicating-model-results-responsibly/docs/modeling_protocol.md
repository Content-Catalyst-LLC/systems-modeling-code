# Communicating Model Results Responsibly Protocol

Model communication should preserve the conditions that give model results meaning.

Core questions:

- What is the model result?
- What assumptions produce the result?
- What uncertainty surrounds the result?
- What boundary limits the result?
- What validation evidence supports the result?
- What are valid and invalid uses?
- What audience is receiving the result?
- What visual or dashboard format could create false precision?
- What distributional effects must be communicated?
- Who remains accountable for decisions made with the model?
