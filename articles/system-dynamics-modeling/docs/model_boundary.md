# Model Boundary

This companion model represents a simplified system dynamics structure.

## Included

- One core stock
- Capacity-limited reinforcing inflow
- Delayed balancing outflow
- Threshold-sensitive correction
- One-time shock
- Scenario comparison
- Sensitivity diagnostics
- Validation checks
- Causal-loop inventory

## Excluded

- Empirical calibration
- Multiple interacting stocks
- Agent heterogeneity
- Network topology
- Event queues
- Geospatial structure
- Institutional decision rules
- Real-world data

## Boundary caution

This model is synthetic and pedagogical. It demonstrates system dynamics architecture and workflow design, not empirical findings.
