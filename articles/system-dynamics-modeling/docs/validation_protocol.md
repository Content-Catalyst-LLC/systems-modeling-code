# Validation Protocol

Validation is purpose-specific. For this synthetic companion model, validation checks workflow integrity and structural plausibility.

## Checks

- Scripts run from a clean checkout.
- Inputs are documented.
- Outputs are reproducible.
- State values remain inside target ranges.
- Longer delays increase overshoot risk.
- Weak balancing produces larger excursions.
- Lower capacity constrains growth.
- Stronger threshold correction limits high-state behavior.
- Sensitivity analysis identifies influential assumptions.
- Model assumptions and limitations are documented.
