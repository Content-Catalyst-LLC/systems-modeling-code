# Model Boundary

The companion model represents a simplified dynamic system for comparing alternative futures.

## Included

- Scenario definitions
- Policy levers
- External shocks
- Dynamic state recurrence
- Resilience buffer
- Stress index
- Policy robustness comparison
- Regret analysis
- Validation checks

## Excluded

- Real climate data
- Real infrastructure data
- Real economic data
- Spatial heterogeneity
- Agent behavior
- Network topology
- Institutional bargaining
- Distributional equity outcomes

## Boundary caution

The model is synthetic. It demonstrates scenario-analysis architecture and workflow design rather than empirical findings.
