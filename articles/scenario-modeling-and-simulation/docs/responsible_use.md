# Responsible Use

Scenario modeling can shape public expectations, investment decisions, climate policy, infrastructure planning, health preparedness, and institutional strategy.

Do not use these synthetic workflows for real decisions without:

- Scenario plausibility review
- Domain validation
- Stakeholder review
- Sensitivity analysis
- Uncertainty communication
- Distributional-impact analysis
- Documentation of exclusions
- Transparent data and code

Scenarios should expand understanding of uncertainty. They should not create false prediction or narrow the future to what is convenient to model.
