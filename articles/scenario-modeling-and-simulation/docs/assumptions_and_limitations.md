# Assumptions and Limitations

## Assumptions

- System state follows a simplified recurrence.
- Policy drag reduces growth pressure.
- Shocks reduce state through a resilience buffer.
- Resilience investment increases system buffer.
- Scenario ensembles vary growth, shock size, and shock timing.
- Regret is calculated relative to the best policy in each synthetic future.

## Limitations

- No empirical calibration is included.
- No probabilities are assigned to named scenarios.
- No real-world decision system is represented.
- Outputs should not be treated as predictions.
- Scenario plausibility is illustrative.
