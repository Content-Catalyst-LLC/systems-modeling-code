# Economic Systems Diagnostics Notes

Economic systems diagnostics should track trajectories, not just endpoints.

## Useful diagnostics

- Output trajectory
- Consumption trajectory
- Investment trajectory
- Capital stock
- Debt stock
- Debt service
- Fragility ratio
- Minimum output after shock
- Maximum fragility
- Scenario comparison
- Validation checks

The key modeling question is not whether one variable changed, but how feedback, accumulation, financial constraints, and policy assumptions changed the system pathway.
