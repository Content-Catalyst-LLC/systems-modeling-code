# Responsible Use

Discrete event simulation can influence staffing, scheduling, service priorities, resource allocation, facility design, and investment decisions.

Do not use these synthetic workflows for real operational decisions without:

- Timestamped data
- Process validation
- Resource validation
- Stakeholder review
- Equity analysis
- Labor and safety review
- Replication and uncertainty reporting
- Sensitivity analysis

DES should clarify operational tradeoffs. It should not hide human consequences behind efficiency metrics.
