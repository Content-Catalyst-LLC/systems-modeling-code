# Model Boundary

This companion folder represents simplified discrete event simulation workflows for queueing and resource-constrained service systems.

## Included

- Future-event-list simulation
- Arrival and departure events
- Single-server and multi-server resources
- Waiting queues
- Stochastic interarrival times
- Stochastic service times
- Queue length tracking
- Resource utilization
- Service-level diagnostics
- Scenario comparison
- Validation checks

## Excluded

- Real operational data
- Multi-stage empirical process maps
- Shift changes and breaks
- Priority classes
- Balking and reneging
- Machine failures in the core queue model
- Downstream blocking
- Optimization over schedules

## Boundary caution

The models are synthetic. They demonstrate DES architecture and workflow design rather than operational findings about a real system.
