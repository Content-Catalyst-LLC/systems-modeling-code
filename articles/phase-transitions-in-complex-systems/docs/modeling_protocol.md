# Phase Transition Modeling Protocol

Phase-transition modeling treats order parameters, control parameters, interaction structure, thresholds, critical regions, and reversibility as explicit analytical objects.

## Core questions

- What macroscopic phase or regime is being studied?
- What variable functions as the order parameter?
- What control parameter drives the transition?
- Which local interactions create collective behavior?
- Is the transition continuous, discontinuous, network-driven, cascade-driven, or hysteretic?
- Where is the critical region?
- Does the system recover along the same path?
- Which early warning signals or structural diagnostics matter?
- How uncertain are the threshold and mechanism?
- What interventions reduce the likelihood of harmful transition?
