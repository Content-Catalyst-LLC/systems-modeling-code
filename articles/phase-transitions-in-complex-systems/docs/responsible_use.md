# Responsible Use

Phase-transition models can clarify threshold risk, collective behavior, and abrupt systemic change. They can also mislead through false precision, overextended metaphor, or deterministic claims.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Mechanism review
- Order-parameter justification
- Control-parameter justification
- Threshold uncertainty analysis
- Sensitivity analysis
- Model comparison
- Stakeholder review
- Distributional impact assessment
- Clear uncertainty communication

A model that claims a phase transition should explain the mechanism that makes the transition plausible.
