# Phase Transition Diagnostics Notes

Phase-transition diagnostics should identify the order parameter, control parameter, transition mechanism, and threshold region.

## Useful diagnostics

- Largest connected component fraction
- Average degree
- Component count
- Approximate giant component probability
- Approximate giant component average degree
- Order-parameter branch behavior
- Critical threshold value
- Hysteresis or reverse-path behavior
- Scenario-level sensitivity

A dramatic curve is not enough. The model should explain why local interactions produce collective reorganization.
