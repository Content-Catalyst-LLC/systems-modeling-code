# Organizational Systems Diagnostics Notes

Organizational diagnostics should track sustainable performance, not only output.

## Useful diagnostics

- Workload trajectory
- Capacity trajectory
- Workload-capacity pressure
- Backlog trajectory
- Burnout trajectory
- Attrition trajectory
- Trust trajectory
- Learning and capability growth
- Coordination burden
- Delivery and quality-risk proxy
- Validation checks

The key modeling question is not whether the organization can produce more output briefly, but whether it can sustain effective, ethical, adaptive performance without hidden human or institutional costs.
