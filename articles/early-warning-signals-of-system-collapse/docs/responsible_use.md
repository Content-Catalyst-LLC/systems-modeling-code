# Responsible Use

Early-warning signals can support prevention, preparedness, and anticipatory governance. They can also be misused through false certainty, panic, complacency, data injustice, or technocratic overreach.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Data audit
- Structural validation
- Multiple indicators
- Window and preprocessing sensitivity analysis
- False-positive and false-negative assessment
- Stakeholder review
- Distributional impact analysis
- Clear response protocols

A missing signal is not proof of safety. A rising signal is not proof of collapse. Early-warning signals should expand responsibility and improve decision-making under uncertainty.
