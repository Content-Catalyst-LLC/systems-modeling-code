# Early Warning Diagnostics Notes

Early-warning indicators are conditional evidence, not deterministic alarms.

## Useful diagnostics

- Rolling variance
- Rolling lag-1 autocorrelation
- Recovery time after perturbation
- Trend slope of rolling indicators
- Final stability parameter
- Maximum absolute system state
- Warning label based on combined variance and autocorrelation behavior
- Spatial clustering examples
- Network fragility examples

Indicators should be interpreted through mechanism, data quality, and system structure. Rising variance alone is not proof of approaching collapse.
