# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Indicator taxonomy is exported.
- Domain, spatial, and network warning examples are exported.
- Scenario trajectories are generated.
- Summary diagnostics are generated.
- State values remain finite.
- Rolling variance remains nonnegative.
- Rolling autocorrelation remains between -1 and 1 when available.
- Trend slopes remain finite.
- Validation checks are exported.

## Professional extension

Applied use should add empirical validation, detrending analysis, seasonal adjustment, model comparison, window sensitivity, false-alarm analysis, missed-warning analysis, and intervention review.
