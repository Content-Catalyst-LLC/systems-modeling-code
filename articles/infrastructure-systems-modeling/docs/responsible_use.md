# Responsible Use

Infrastructure systems models can influence public safety, utility investment, transportation access, climate adaptation, emergency response, public finance, service affordability, land development, and community protection. They should be used with transparency and accountability.

Do not use these synthetic workflows for real decisions without:

- Engineering data
- Local operations data
- Dependency audit
- Calibration against observed disruptions
- Validation and stress testing
- Equity assessment
- Critical-user analysis
- Community participation
- Clear communication of limitations

Infrastructure systems modeling should make public consequences more visible. It should not hide contested investment, restoration, or protection priorities behind technical language.
