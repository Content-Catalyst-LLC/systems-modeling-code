# Validation Protocol

Validation evaluates whether the calibrated model is credible for its intended purpose.

## Companion checks

- Calibration RMSE
- Validation RMSE
- Calibration MAE
- Validation MAE
- Bias
- Generalization gap
- Parameter plausibility
- Residual diagnostics
- Benchmark comparison
- Structural validation inventory

## Professional extension

Applied validation should include:

- Independent data where possible
- Out-of-sample validation
- Structural review
- Extreme-condition tests
- Sensitivity analysis
- Benchmark comparison
- Stakeholder or expert review
- Clear domain of applicability
