# Credibility Evidence

Model credibility is purpose-specific. A model can be useful for explanation while inappropriate for precise forecasting.

## Evidence categories

- Verification evidence
- Calibration evidence
- Empirical validation evidence
- Structural validation evidence
- Out-of-sample performance
- Sensitivity and robustness evidence
- Benchmark comparison
- Documentation quality

## Companion warning

The workflows in this folder use synthetic data. They show how credibility evidence can be organized, not whether any real system has been validated.
