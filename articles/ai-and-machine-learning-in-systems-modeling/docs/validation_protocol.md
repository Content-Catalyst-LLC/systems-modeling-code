# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and numerical behavior.

## Checks

- AI role taxonomy is exported.
- Hybrid architecture taxonomy is exported.
- Governance, data risk, and constraint tables are exported.
- Scenario predictions are generated.
- Baseline and hybrid error metrics are generated.
- Hybrid improvement is calculated.
- Validation checks are exported.
- Optional language workflows run when compilers/interpreters are available.

## Professional extension

Applied workflows should add cross-validation, out-of-distribution testing, subgroup performance analysis, uncertainty intervals, causal review, privacy assessment, security review, human oversight protocols, drift monitoring, and post-deployment evaluation.
