# AI Systems Modeling Diagnostics Notes

The central diagnostic is not whether the hybrid model improves a metric once. The central diagnostic is whether the learning component improves performance for valid reasons while preserving interpretability, constraints, and governance.

## Useful diagnostics

- Baseline RMSE and MAE
- Hybrid RMSE and MAE
- Hybrid improvement ratio
- Error by scenario
- Residual magnitude
- Drift sensitivity
- Constraint violations
- Data risk review
- Governance readiness
- Valid operating domain

A hybrid model should be interpreted as a coupled system: structural assumptions plus learned empirical correction.
