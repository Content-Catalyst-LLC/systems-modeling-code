# Health Systems Modeling Protocol

Health systems modeling treats population, health need, care pathways, service capacity, workforce, behavior, trust, financing, incentives, public health, social determinants, equity, data systems, and governance as explicit analytical objects.

## Core questions

- What health system question is the model designed to examine?
- Which population, service, organization, community, or policy boundary is included?
- Which care pathway or public health pathway is being represented?
- Which demand, capacity, backlog, and workforce constraints matter?
- Which feedback loops amplify strain or improve outcomes?
- Which social determinants and access barriers shape results?
- Which equity dimensions must be disaggregated?
- Which uncertainty and scenario assumptions drive conclusions?
- Which privacy, consent, bias, and clinical governance issues must be addressed?
- How should clinicians, public health practitioners, patients, and communities review interpretation?

## Companion workflow

The companion workflows compare baseline health system, higher demand growth, stronger prevention, larger surge, faster hiring, higher access barrier, burnout-sensitive, and resilient prepared system scenarios.
