# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Health system component taxonomy is exported.
- Feedback-loop taxonomy is exported.
- Modeling approaches and equity dimensions are exported.
- Scenario trajectories are generated.
- Summary diagnostics are generated.
- Trust remains within expected bounds.
- Demand, capacity, backlog, pressure, burnout, attrition, served volume, unmet need, and access gap remain nonnegative.
- Validation checks are exported.

## Professional extension

Applied use should add patient-flow data, public health data, staffing records, access and equity indicators, privacy review, clinical validation, community review, uncertainty intervals, and scenario stress testing.
