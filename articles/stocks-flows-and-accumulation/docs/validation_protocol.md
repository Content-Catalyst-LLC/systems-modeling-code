# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Stock-flow taxonomy is exported.
- Scenario definitions are exported.
- Trajectories are generated for each scenario.
- Summary diagnostics are generated for each stock.
- Backlog remains nonnegative.
- Resource remains nonnegative.
- Infrastructure condition remains between 0 and 100.
- Net flows remain finite.
- Validation tables are exported.

## Professional extension

Applied use should add empirical calibration, dimensional review, historical behavior reproduction, sensitivity analysis, uncertainty propagation, model comparison, and distributional impact assessment.
