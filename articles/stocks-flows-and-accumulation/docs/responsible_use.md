# Responsible Use

Stock-flow models can make accumulated harm visible. They can also hide responsibility if accumulation is presented as impersonal or inevitable.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Boundary review
- Dimensional checking
- Structural validation
- Parameter justification
- Sensitivity and uncertainty analysis
- Distributional review
- Stakeholder interpretation where affected communities are involved
- Clear communication of what is and is not modeled

Stock-flow analysis should make accumulated consequences more accountable, not less.
