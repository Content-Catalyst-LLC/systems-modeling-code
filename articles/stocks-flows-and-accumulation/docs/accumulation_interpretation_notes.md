# Accumulation Interpretation Notes

A stock changes only when net flow changes it.

## Interpretation rules

- If inflow exceeds outflow, the stock grows.
- If outflow exceeds inflow, the stock shrinks.
- If inflow and outflow balance, the stock stabilizes.
- Reducing an inflow does not necessarily reduce the stock.
- Increasing an outflow does not necessarily produce immediate recovery.
- Delays can allow pressure to accumulate before corrective action arrives.
- Capacity limits can make recovery slower than policy narratives imply.

The central diagnostic is sustained net flow, not one-time effort.
