# Assumptions and Limitations

## Assumptions

- Edges are undirected in core examples.
- Edge weights represent generic relationship strength.
- Capacity is illustrative.
- Contagion uses a simple probabilistic rule.
- Cascades use a simple neighborhood threshold rule.
- Targeted removal uses degree ranking.

## Limitations

- No empirical data are used.
- No real network is represented.
- Edge timing is static.
- Edge direction is not represented.
- Functional capacity loss is simplified.
- Centrality measures are limited to dependency-light diagnostics.
- Outputs should not be interpreted as real-world findings.
