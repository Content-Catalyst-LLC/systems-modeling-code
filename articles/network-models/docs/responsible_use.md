# Responsible Use

Network models can expose sensitive relationships and systemic vulnerabilities.

Do not apply these workflows to real people, institutions, critical infrastructure, financial systems, or supply chains without:

- Data governance
- Privacy review
- Access controls
- Missing-edge assessment
- Domain validation
- Stakeholder review
- Security-sensitive redaction
- Uncertainty communication

A network model should clarify interdependence. It should not disclose sensitive structure or create false authority.
