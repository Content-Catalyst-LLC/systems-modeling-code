# Model Boundary

This companion model represents a synthetic multilayer-style network with infrastructure, logistics, and digital node groups.

## Included

- Nodes and edges
- Edge weights and capacities
- Local edges
- Bridge edges
- Cross-layer edges
- Degree diagnostics
- Component diagnostics
- Path-length diagnostics
- Random node removal
- Targeted high-degree removal
- Simple contagion simulation
- Threshold cascade simulation

## Excluded

- Real infrastructure data
- Real financial exposure data
- Real public-health contact data
- Temporal edge changes
- Directed flow constraints
- Full load redistribution
- Recovery scheduling
- Privacy-sensitive relationship data

## Boundary caution

The network is synthetic. It demonstrates methods and workflow structure rather than empirical claims about a real system.
