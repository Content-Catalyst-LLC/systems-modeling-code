# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Logistic-map states remain bounded.
- Trajectory differences remain finite.
- Network node and edge tables are exported.
- Degree diagnostics remain plausible.
- Network diffusion trajectories are exported.
- Entropy values remain finite.
- Dispersion diagnostics remain finite.
- Summary tables are generated.
