# Assumptions and Limitations

## Assumptions

- All data are synthetic.
- The logistic map is used as an illustrative nonlinear recurrence.
- The network is a synthetic dependency graph.
- Diffusion is represented by neighbor averaging.
- Stochastic shocks are normally distributed with fixed variance.
- Entropy diagnostics use simple binned state distributions.

## Limitations

- No empirical system is represented.
- No real-world network is calibrated.
- No formal stability proof is provided.
- No domain-specific validation is performed.
- Results should not be used as real-world decision guidance.
