# Responsible Use

Complex systems mathematics can make weak assumptions look authoritative if formal notation is not paired with clear interpretation.

Do not use these synthetic workflows for real decisions without:

- Domain data
- Boundary review
- Structural validation
- Parameter justification
- Uncertainty analysis
- Sensitivity testing
- Stakeholder review when public consequences are involved
- Clear communication of model limits

Mathematical precision is not the same as real-world certainty.
