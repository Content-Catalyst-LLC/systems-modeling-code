# Validation Protocol

Synthetic validation checks:

- Policy table is generated.
- Scenario table is generated.
- Scenario result count equals policy count times scenario count.
- Composite scores are finite.
- Regret values are nonnegative.
- Metric weights sum to one.
- Policy summary table is generated.

Applied validation would require empirical evidence, expert review, stakeholder review, implementation evidence, and monitoring against observed outcomes.
