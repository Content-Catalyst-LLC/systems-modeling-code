# Responsible Use

Use this companion folder to understand agent-based modeling concepts and reproducible workflow design.

Do not use it for decisions about real communities, markets, epidemics, or institutions without:

- Domain data
- Empirical calibration
- Behavioral validation
- Stakeholder review
- Privacy safeguards
- Uncertainty communication
- Distributional analysis
- Ethical interpretation

ABM can appear realistic because it models individual agents. That realism can be misleading when rules are synthetic or weakly justified.
