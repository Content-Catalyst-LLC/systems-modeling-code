# Model Boundary

This companion folder contains two synthetic ABM structures:

1. A Schelling-style spatial relocation model.
2. A threshold-adoption model on a ring neighborhood.

## Included

- Heterogeneous agents
- Agent state variables
- Local interaction
- Spatial relocation
- Threshold adoption
- Scenario comparison
- Repeated simulation logic
- Emergence diagnostics
- Validation checks
- ODD-style documentation notes

## Excluded

- Empirical calibration
- Real population data
- Real household mobility data
- Real contact networks
- Network rewiring
- Learning beyond threshold response
- Institutional rule enforcement
- Privacy-sensitive data

## Boundary caution

The models are pedagogical and synthetic. They demonstrate ABM architecture and workflow design rather than real-world social findings.
