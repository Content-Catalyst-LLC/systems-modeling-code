# Threshold Diagnostics Notes

Threshold diagnostics should distinguish ordinary variation from structural transition risk.

## Useful diagnostics

- First degraded-regime time
- Degraded-period count
- Final regime
- Minimum system state
- Maximum pressure
- Mean net flow
- Collapse threshold
- Recovery threshold
- Hysteresis gap
- Rolling variance
- Rolling autocorrelation

Threshold location is often uncertain. Model ranges are usually more honest than one exact boundary.
