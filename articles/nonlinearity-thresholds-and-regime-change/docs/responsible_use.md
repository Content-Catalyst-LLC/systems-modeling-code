# Responsible Use

Nonlinear threshold models can support precaution, resilience planning, and early intervention. They can also be misused through dramatic claims, false precision, or technocratic overreach.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Boundary review
- Threshold uncertainty analysis
- Structural validation
- Sensitivity analysis
- Model comparison
- Stakeholder review
- Distributional impact assessment
- Clear communication of uncertainty and limits

Threshold models should help decision-makers act earlier and more responsibly without pretending uncertainty has disappeared.
