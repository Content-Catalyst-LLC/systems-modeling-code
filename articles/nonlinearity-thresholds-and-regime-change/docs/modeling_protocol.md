# Nonlinearity, Threshold, and Regime-Change Modeling Protocol

Nonlinear threshold modeling treats state-dependent response, saturation, collapse, recovery thresholds, and regime-specific behavior as core system structure.

## Core questions

- Which system response is nonlinear?
- Which stocks accumulate pressure?
- Which thresholds may change behavior?
- Are thresholds precise, uncertain, or contested?
- Does the model need multiple regimes?
- Does recovery follow the same path as degradation?
- Are there early-warning signals of resilience loss?
- Do interacting stressors create compound nonlinear risk?
- Do conclusions depend on one nonlinear functional form?

## Companion workflow

The companion workflows compare early intervention, late intervention, strong recovery, lower-threshold stress, high-resilience buffering, hysteresis trap, compound pressure, and rapid prevention scenarios.
