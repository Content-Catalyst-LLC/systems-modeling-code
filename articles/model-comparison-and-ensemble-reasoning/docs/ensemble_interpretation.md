# Ensemble Interpretation

Ensemble reasoning should not automatically collapse disagreement into an average.

## Interpretive checks

- Report individual model outputs.
- Report ensemble mean or weighted ensemble only after model-specific diagnostics.
- Compare validation performance.
- Record model-dependence notes.
- Report spread, ranking, and residual diagnostics.
- Use regret and robustness when decisions are involved.

## Warning

An ensemble range is not automatically a probability distribution. It only represents the uncertainty space included in the ensemble design.
