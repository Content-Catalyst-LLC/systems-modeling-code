# Responsible Use

Model comparison and ensemble reasoning can create false confidence if ensembles are presented as more independent, complete, or probabilistic than they are.

Do not use these synthetic workflows for real decisions without:

- Empirical validation data
- Justified model selection
- Documented model dependence
- Structural review
- Sensitivity to weighting rules
- Scenario design review
- Stakeholder review where decisions affect communities
- Clear uncertainty communication

A model ensemble should make uncertainty more visible, not bury it beneath an average.
