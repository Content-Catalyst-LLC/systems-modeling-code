# Responsible Use

A model should not be used as a substitute for judgment, accountability, public reasoning, or ethical review.

Do not use this synthetic workflow for real decisions without:

- Model purpose definition
- Data quality review
- Assumption register
- Boundary critique
- Sensitivity analysis
- Validation for intended use
- Stakeholder review where affected groups are involved
- Communication controls
- Valid-use and invalid-use documentation
- Accountability for decisions made with the model

The correct question is not whether the model is true in general. It is whether the model is useful, valid, and responsibly communicated for a specific purpose.
