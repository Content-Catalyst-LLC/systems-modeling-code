# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and bounded score behavior.

## Checks

- Model cases are generated.
- Clarification scores remain between zero and one.
- Distortion risk scores remain between zero and one.
- Risk register is exported.
- Communication controls are exported.
- Use-scope register is exported.
- Distortion patterns are exported.

## Professional extension

Applied validation should include model purpose review, structural validation, data validation, sensitivity analysis, boundary sensitivity, stakeholder review, communication review, and governance controls.
