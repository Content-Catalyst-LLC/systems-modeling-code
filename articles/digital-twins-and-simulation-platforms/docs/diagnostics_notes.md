# Digital Twin Diagnostics Notes

Digital twin diagnostics should evaluate state tracking, anomaly detection, intervention triggers, synchronization, and governance readiness.

## Useful diagnostics

- MAE of noisy observations versus true state
- MAE of twin estimate versus true state
- RMSE of noisy observations versus true state
- RMSE of twin estimate versus true state
- Tracking improvement ratio
- Residual trajectory
- Anomaly count
- Intervention count
- Scenario comparison
- Validation checks
- Governance and security readiness

The key question is not whether the twin has live data, but whether the live data improve state awareness, forecast reliability, and decision quality within a governed operating domain.
