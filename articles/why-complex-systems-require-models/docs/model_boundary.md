# Model Boundary

This companion model represents a simplified dynamic system with reinforcing growth, delayed balancing feedback, threshold-sensitive correction, and a discrete shock.

## Included

- Reinforcing growth
- Delayed corrective response
- Threshold-sensitive penalty
- One-time shock
- Scenario parameters
- Summary diagnostics
- Sensitivity tests
- Validation checks

## Excluded

- Empirical calibration
- Multi-agent behavior
- Spatial dynamics
- Network propagation
- Institutional decision-making
- Stakeholder values
- Real-world data

## Boundary caution

The model is designed to demonstrate why complex systems require formal representation. It is not a real-world model of climate, infrastructure, ecology, finance, or policy.
