# Panarchy Diagnostics Notes

Panarchy diagnostics should distinguish phase behavior, cross-scale coupling, and memory effects.

## Useful diagnostics

- Final fast-cycle state
- Final slow-memory state
- Release-event count
- Maximum fast-cycle state
- Maximum slow-memory state
- Mean cross-scale coupling
- Growth periods
- Conservation periods
- Release periods
- Reorganization periods
- Scenario-specific diagnostic label

Panarchy should not be reduced to a story about four phases. The model should define the variables and transition rules that make phase classification meaningful.
