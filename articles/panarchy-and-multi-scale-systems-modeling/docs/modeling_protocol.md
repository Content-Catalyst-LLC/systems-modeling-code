# Panarchy and Multi-Scale Systems Modeling Protocol

Panarchy modeling treats nested scales, fast variables, slow variables, adaptive-cycle phases, revolt, remember, and cross-scale coupling as explicit system structure.

## Core questions

- What is the focal system?
- What lower and higher scales influence it?
- Which variables change quickly?
- Which variables change slowly but structure possibility?
- How are growth, conservation, release, and reorganization represented?
- When does lower-scale disturbance trigger higher-scale change?
- How does higher-scale memory shape reorganization?
- Does the system recover, transform, or become trapped?
- Which scale defines success, and who benefits?

## Companion workflow

The companion workflows compare baseline panarchy, strong revolt, strong remember, rigid slow structure, weak memory with high volatility, adaptive transformation, locked conservation, and recurrent release scenarios.
