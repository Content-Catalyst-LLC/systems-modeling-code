# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Panarchy concept taxonomy is exported.
- Multi-scale examples are exported.
- Scenario trajectories are generated.
- Summary diagnostics are generated.
- Fast-cycle and slow-memory values remain nonnegative and finite.
- Release-event counts remain nonnegative.
- Phase-period counts remain nonnegative.
- Cross-scale coupling remains nonnegative and finite.
