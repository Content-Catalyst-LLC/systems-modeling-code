# Assumptions and Limitations

## Assumptions

- All data are synthetic.
- Fast-cycle state is a stylized representation of a lower-scale adaptive cycle.
- Slow-memory state is a stylized representation of higher-scale memory or constraint.
- Release occurs when the fast-cycle state crosses a threshold.
- Revolt increases slow-memory state after release.
- Remember stabilizes or constrains fast-cycle reorganization.
- Phase labels are illustrative diagnostics.

## Limitations

- No empirical ecological, infrastructure, economic, health, organizational, climate, or governance system is calibrated.
- Only two scale levels are modeled in the core workflow.
- Real panarchy dynamics may include many nested scales and heterogeneous actors.
- Phase labels are simplified.
- Applied use requires domain data, stakeholder review, validation, and explicit scale-boundary judgment.
