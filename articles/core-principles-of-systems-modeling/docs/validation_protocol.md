# Validation Protocol

Validation is purpose-specific. For this synthetic companion model, validation means checking workflow integrity and interpretable model behavior.

## Checks

- Scripts run from a clean checkout.
- Inputs are documented.
- Outputs are reproducible.
- State remains within synthetic target ranges.
- Longer delays increase overshoot risk.
- Lower capacity changes saturation behavior.
- Stronger threshold correction constrains high-state behavior.
- Sensitivity analysis identifies influential assumptions.
- Model assumptions and limitations are documented.

## Professional extensions

- Calibrate to empirical data.
- Compare multiple model structures.
- Add stakeholder review.
- Add uncertainty ensembles.
- Add formal model cards and reproducibility reports.
