# Model Boundary

This companion model represents the core principles of systems modeling in a simplified stock-flow system.

## Included

- Stock accumulation
- Reinforcing inflow
- Delayed balancing outflow
- Capacity-limited growth
- Threshold-sensitive correction
- Discrete shock
- Scenario comparison
- Sensitivity diagnostics
- Validation checks
- Principle inventory

## Excluded

- Empirical calibration
- Multiple interacting stocks
- Agent heterogeneity
- Network propagation
- Spatial structure
- Institutional decision rules
- Real-world data

## Boundary caution

The model is pedagogical and synthetic. It demonstrates modeling structure rather than real-world evidence.
