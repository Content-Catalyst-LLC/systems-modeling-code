# Systems Modeling

This article folder supports reproducible examples for formal systems modeling.

## Topics

- system boundaries
- stocks and flows
- reinforcing feedback
- balancing feedback
- time delays
- nonlinear interaction
- network shock propagation
- scenario simulation
- sensitivity analysis
- model assumptions
- calibration and validation concepts
- responsible model interpretation

## Folder Structure

- `python/` — stock-flow simulation, network shock propagation, scenario analysis, assumption risk
- `r/` — interacting stocks, feedback loops, scenario summaries
- `julia/` — compact systems simulation examples
- `sql/` — systems model metadata, components, links, assumptions, runs, and outputs
- `c/`, `cpp/`, `fortran/`, `rust/`, `go/` — compact stock-flow examples
- `docs/` — modeling notes and interpretation guidance
- `data/` — synthetic teaching inputs and model parameters
- `outputs/` — generated outputs
- `notebooks/` — notebook placeholders

## Modeling Warning

These examples are educational. Real systems modeling should evaluate system boundaries, evidence quality, parameter credibility, feedback structure, uncertainty, sensitivity, calibration, validation, stakeholder context, and model purpose.
