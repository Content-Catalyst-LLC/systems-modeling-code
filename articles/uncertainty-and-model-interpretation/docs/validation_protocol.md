# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Scenario inventory is generated.
- Policy ensemble outputs are generated.
- Resilience scores remain between 0 and 100.
- Regret is nonnegative.
- State variables remain nonnegative.
- Cumulative stress remains nonnegative.
- Summary tables preserve policy-level diagnostics.
- Confidence notes and robustness interpretations are exported.
