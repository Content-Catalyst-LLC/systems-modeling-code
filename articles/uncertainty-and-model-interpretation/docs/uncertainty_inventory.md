# Uncertainty Inventory

This companion folder distinguishes several uncertainty sources rather than treating uncertainty as a single generic error term.

## Included uncertainty types

- Parameter uncertainty
- Scenario uncertainty
- Structural uncertainty
- Data and measurement uncertainty
- Stochastic uncertainty
- Value and interpretation uncertainty
- Deep uncertainty

## Professional extension

Applied modeling should document which uncertainties are quantified, which are scenario-based, which are structural, which are qualitative, and which remain unresolved.
