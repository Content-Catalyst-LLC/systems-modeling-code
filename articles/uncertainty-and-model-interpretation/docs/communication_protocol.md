# Communication Protocol

Uncertainty communication should separate:

- Model assumptions
- Parameter ranges
- Scenario conditions
- Ensemble summaries
- Confidence language
- Robustness findings
- Fragility findings
- Domain of applicability
- Limits of interpretation

## Recommended summary format

1. What the model can support
2. What the model cannot support
3. Which assumptions matter most
4. Which conclusions are robust
5. Which conclusions are fragile
6. How uncertainty should affect decisions
