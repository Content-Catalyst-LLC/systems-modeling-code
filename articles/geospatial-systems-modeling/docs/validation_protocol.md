# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Spatial component taxonomy is exported.
- Spatial data structure taxonomy is exported.
- Scale, boundary, and ethics registers are exported.
- Scenario grids are generated.
- Population is nonnegative.
- Hazard and vulnerability remain in the unit interval.
- Risk and accessibility scores are nonnegative.
- Priority zones are generated.
- Validation checks are exported.

## Professional extension

Applied geospatial validation should add coordinate reference system checks, topology checks, projection management, ground-truth review, sensitivity to boundary and scale, routing validation, uncertainty surfaces, privacy review, and participatory review where communities are affected.
