# Geospatial Diagnostics Notes

Geospatial diagnostics should evaluate both spatial model behavior and the assumptions that produce mapped outputs.

## Useful diagnostics

- Cell count and grid completeness
- Population distribution
- Hazard and vulnerability ranges
- Total and average risk score
- Accessibility score
- Service gap score
- Priority-zone counts
- Scenario comparison
- Scale and boundary sensitivity
- Data quality and uncertainty review
- Spatial equity and ethics review

A map is not a model by itself. The model is the spatial data structure, relationship assumptions, scoring logic, uncertainty handling, validation, and interpretation behind the map.
