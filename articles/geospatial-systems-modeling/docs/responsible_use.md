# Responsible Use

Geospatial systems models can influence infrastructure investment, environmental regulation, public health response, climate adaptation, policing, insurance, zoning, disaster planning, service allocation, and public attention. Spatial outputs can shape how institutions see places.

Do not use these synthetic workflows for real decisions without:

- Real geospatial data governance
- Coordinate reference system and projection checks
- Metadata and data lineage
- Privacy review
- Scale and boundary sensitivity testing
- Uncertainty communication
- Local and domain validation
- Spatial equity review
- Participatory review where appropriate
- Clear limits on interpretation

Geospatial models should clarify spatial systems. They should not disguise uncertainty, stigmatize places, erase local knowledge, or convert incomplete data into false objectivity.
