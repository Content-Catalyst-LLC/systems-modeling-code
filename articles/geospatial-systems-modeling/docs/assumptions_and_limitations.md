# Assumptions and Limitations

## Assumptions

- All data are synthetic.
- Spatial units are simple grid cells.
- Distance is Euclidean, not road-network travel time.
- Hazard is represented as proximity to a synthetic river-like line.
- Population and vulnerability are stylized surfaces.
- Accessibility uses distance-decay weighting.
- Priority zones are threshold-based.
- Governance and ethics tables are conceptual.

## Limitations

- This is not a GIS workflow using real coordinate reference systems.
- It does not perform real spatial joins, projections, routing, topology checks, raster processing, or network analysis.
- It does not represent a real city, watershed, infrastructure system, environmental exposure, or service geography.
- Applied use requires geospatial libraries, validated data, metadata, projection management, uncertainty analysis, privacy review, local knowledge, and domain validation.
