# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Environmental component taxonomy is exported.
- Feedback-loop taxonomy is exported.
- Modeling approaches and justice dimensions are exported.
- Resource-stock trajectories are generated.
- Pollution-exposure trajectories are generated.
- Summary diagnostics are generated.
- Stock, concentration, exposure, and cumulative burden remain nonnegative.
- Validation checks are exported.

## Professional extension

Applied use should add monitoring-data comparison, spatial validation, source inventory review, hydrological or atmospheric validation, historical event comparison, sensitivity analysis, uncertainty intervals, environmental justice review, and participatory interpretation.
