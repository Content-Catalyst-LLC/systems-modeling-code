# Responsible Use

Environmental systems models can influence regulation, permitting, cleanup, conservation, land-use planning, climate adaptation, public health, and infrastructure investment. They should be used with transparency and humility.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Monitoring data
- Source and pathway audit
- Model calibration
- Validation against observed behavior
- Uncertainty and sensitivity analysis
- Distributional impact assessment
- Community participation
- Clear communication of limitations

Environmental systems modeling should make exposure, uncertainty, cumulative burden, and responsibility more visible. It should not hide contested choices behind technical authority.
