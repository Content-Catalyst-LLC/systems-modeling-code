# Environmental Systems Diagnostics Notes

Environmental diagnostics should track trajectories, pathways, exposure, and recovery rather than only endpoint indicators.

## Useful diagnostics

- Environmental stock trajectory
- Regeneration and extraction rates
- Restoration rate
- Disturbance effects
- Resilience index
- Pollutant concentration
- Active pollutant load
- Decay and flow loss
- Exposure and cumulative exposure
- Intervention status
- Scenario comparison
- Validation checks

The key modeling question is not only whether environmental quality changes, but which pathways, accumulations, feedbacks, and interventions generate that change.
