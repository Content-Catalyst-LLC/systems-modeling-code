# Responsible Use

Assumption and boundary documentation can influence public policy, infrastructure planning, climate adaptation, environmental management, health systems, AI governance, and organizational decision-making.

Do not use these synthetic workflows for real decisions without:

- Domain review
- Evidence grading
- Assumption register ownership
- Boundary critique
- Stakeholder review
- Sensitivity analysis
- Validation
- Clear communication of valid and invalid uses
- Documentation of excluded consequences
- Governance for updates and model use

A model with visible assumptions is more trustworthy than a model that pretends to have none.
