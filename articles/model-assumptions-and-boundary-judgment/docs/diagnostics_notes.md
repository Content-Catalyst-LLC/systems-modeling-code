# Diagnostics Notes

Assumption diagnostics should identify fragile conclusions before model outputs are used in decisions.

## Useful diagnostics

- Assumption risk score
- High-risk assumption count
- Assumption category summary
- Boundary scenario ranking
- Difference between narrow and expanded boundaries
- Exclusion log review
- Evidence strength grading
- Validation checks
- Stakeholder review status
- Contested assumption count

The goal is not to eliminate assumptions. The goal is to make them visible, reviewable, testable, and proportionate to the model purpose.
