# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and bounded score behavior.

## Checks

- Assumption register is generated.
- Risk scores are between zero and one.
- Boundary scenarios are generated.
- Composite scores are between zero and one.
- Exclusion log is exported.
- Boundary critique questions are exported.
- Evidence strength table is exported.

## Professional extension

Applied validation should add formal sensitivity analysis, calibration evidence, independent review, boundary sensitivity tests, stakeholder critique, extreme-condition tests, and model-use governance.
