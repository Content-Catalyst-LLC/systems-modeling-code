# Assumptions and Limitations

## Assumptions

- All data are synthetic.
- Assumption risk is represented as uncertainty × sensitivity × consequence.
- Boundary scenarios use normalized scores from zero to one.
- Composite boundary scores are illustrative.
- Evidence strength categories are simplified.
- Exclusion logs are representative examples.

## Limitations

- This workflow does not perform formal uncertainty quantification.
- It does not replace domain validation, stakeholder review, or ethical boundary critique.
- It does not prove which boundary is correct.
- Applied use requires evidence review, stakeholder participation, version control, model validation, sensitivity analysis, and responsible communication.
