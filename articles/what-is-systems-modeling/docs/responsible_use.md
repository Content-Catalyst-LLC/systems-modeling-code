# Responsible Use

Systems models can improve reasoning, but they can also create false confidence.

Use this code to:

- Clarify assumptions
- Compare scenarios
- Test sensitivity
- Build reproducible modeling workflows
- Teach model structure
- Prototype validation diagnostics

Do not use this code to:

- Make real decisions without empirical calibration
- Hide uncertainty
- Imply false precision
- Exclude affected stakeholders
- Optimize one metric while ignoring harms elsewhere
- Treat synthetic outputs as evidence about real systems
