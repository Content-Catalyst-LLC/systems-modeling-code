# Assumptions and Limitations

This folder contains synthetic models for professional workflow demonstration.

## Assumptions

- System state can be represented at discrete time steps.
- Network dependency can be represented by weighted directed edges.
- Shock propagation is shaped by coupling, redundancy, recovery, and topology.
- Stock-flow behavior can be represented through recurrence equations.
- Monte Carlo ensembles can reveal parameter sensitivity.
- Validation targets can be used as plausibility checks.

## Limitations

- The data are synthetic.
- Parameters are illustrative.
- No empirical calibration is included.
- Social, institutional, ecological, and political dynamics are simplified.
- Results should not be interpreted as real-world findings.
- Model validity depends on purpose, evidence, and boundary judgment.
