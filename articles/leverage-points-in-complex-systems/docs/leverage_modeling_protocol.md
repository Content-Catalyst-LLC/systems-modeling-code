# Leverage-Point Modeling Protocol

Leverage-point modeling compares interventions by depth and system effect.

## Core questions

- What persistent behavior is the system reproducing?
- Which stocks, flows, feedback loops, delays, rules, and goals generate that behavior?
- Which intervention changes only a parameter?
- Which intervention changes feedback structure?
- Which intervention changes information flow?
- Which intervention changes rules, authority, or incentives?
- Which intervention changes self-organization or learning capacity?
- Which intervention changes system goals or paradigms?
- Which intervention remains effective under stress and uncertainty?

## Companion workflow

The companion workflows compare baseline, parameter, buffer, delay, feedback, information-flow, rule, self-organization, goal, and delayed-deep interventions.
