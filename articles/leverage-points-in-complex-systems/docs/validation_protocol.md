# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Leverage hierarchy inventory is exported.
- Intervention trajectories are generated.
- Summary diagnostics are exported.
- State and pressure remain nonnegative and finite.
- Resilience and learning capacity remain bounded.
- Cumulative intervention remains nonnegative.
- Leverage ratio remains finite.
- Domain examples are exported.

## Professional extension

Applied use should add empirical validation, stakeholder review, stress testing, model comparison, uncertainty propagation, sensitivity analysis, and distributional impact analysis.
