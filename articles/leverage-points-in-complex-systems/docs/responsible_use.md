# Responsible Use

Leverage-point analysis can justify powerful interventions. Calling something a leverage point does not make it legitimate.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Boundary review
- Structural validation
- Parameter justification
- Sensitivity and uncertainty analysis
- Stakeholder review
- Distributional impact analysis
- Governance legitimacy
- Clear communication of model limits

Leverage analysis should make intervention power more accountable, not less.
