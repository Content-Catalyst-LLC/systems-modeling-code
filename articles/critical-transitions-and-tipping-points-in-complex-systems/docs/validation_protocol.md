# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Tipping-mechanism taxonomy is exported.
- Domain examples are exported.
- Scenario trajectories are generated.
- Summary diagnostics are generated.
- State values remain finite.
- Rolling variance is nonnegative.
- Rolling autocorrelation remains between -1 and 1 when available.
- Approximate transition steps remain nonnegative.
- Hysteresis gap remains nonnegative when detected.

## Professional extension

Applied use should add empirical validation, threshold uncertainty, structural model comparison, external forcing scenarios, noise sensitivity, network cascade modeling, distributional impact assessment, and governance review.
