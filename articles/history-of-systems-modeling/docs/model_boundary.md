# Model Boundary

This companion folder models historical structures, not historical institutions.

## Included

- Exponential growth
- Logistic constraint
- Delayed feedback regulation
- Shock response
- Scenario comparison
- Sensitivity diagnostics
- Historical-method inventory
- Validation checks

## Excluded

- Empirical historical data
- Full reconstructions of original historical models
- Real Club of Rome or World3 model logic
- Agent-based simulation
- Network propagation
- Integrated assessment calibration
- Digital twin architecture

## Boundary caution

The models are pedagogical and synthetic. They demonstrate how historical modeling ideas can be formalized in reproducible code.
