# Historical Modeling Notes

The article argues that systems modeling evolved as researchers gained better ways to represent complex system structure.

## Core historical progression

1. Feedback control
2. General system organization
3. Stock-flow accumulation
4. Delayed feedback
5. Computer simulation
6. Global coupled-system modeling
7. Agent-based and network modeling
8. Integrated assessment modeling
9. Hybrid modeling and digital twins

## Modeling lesson

Each historical tradition added representational capacity. The point is not that one method replaces the others. The point is that different forms of complexity require different modeling structures.
