# Responsible Use

Cascade and contagion models can reveal hidden systemic risk. They can also mislead through false precision, blame displacement, stigmatization, excessive control, or overextended metaphor.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Network and dependency audit
- Threshold uncertainty analysis
- Mechanism review
- Model comparison
- Sensitivity testing
- Stakeholder review
- Distributional impact assessment
- Clear uncertainty communication

Cascade modeling should make system fragility and responsibility more visible. It should not be used to stigmatize people or make systemic harm appear inevitable.
