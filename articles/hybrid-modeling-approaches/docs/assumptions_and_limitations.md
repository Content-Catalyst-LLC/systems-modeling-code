# Assumptions and Limitations

## Assumptions

- Agent demand propensities are synthetic.
- Queue pressure is a simple ratio of backlog to capacity.
- Feedback from pressure to demand occurs with a one-step delay.
- Aggregate adoption feedback is simplified.
- Interfaces exchange scalar summary variables.
- Scenarios are synthetic and illustrative.

## Limitations

- No empirical calibration is included.
- No real agent population is represented.
- No real operational queue is represented.
- No real network is represented in the core simulation.
- No uncertainty propagation beyond scenario and ensemble examples.
- Results should not be interpreted as predictions.
