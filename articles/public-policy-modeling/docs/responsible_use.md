# Responsible Use

Public policy models can influence law, budgets, enforcement, services, benefits, taxes, infrastructure, environmental protection, health, safety, rights, and community life. They should support public reasoning, not replace it.

Do not use these synthetic workflows for real decisions without:

- Domain evidence
- Legal and institutional context
- Implementation review
- Calibration and validation
- Sensitivity and uncertainty analysis
- Equity assessment
- Community participation
- Clear communication of limitations

Public policy modeling should make public consequences more visible. It should not hide democratic choices behind technical authority.
