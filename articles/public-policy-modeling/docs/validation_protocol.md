# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Policy component taxonomy is exported.
- Feedback-loop taxonomy is exported.
- Modeling approaches and equity dimensions are exported.
- Scenario trajectories are generated.
- Summary diagnostics are generated.
- Trust and uptake remain within expected bounds.
- System state, capacity, burden, policy intensity, and side effects remain nonnegative.
- Validation checks are exported.

## Professional extension

Applied use should add real policy data, program records, legal constraints, administrative workflow data, implementation evidence, historical comparison, sensitivity analysis, uncertainty intervals, equity review, and participatory interpretation.
