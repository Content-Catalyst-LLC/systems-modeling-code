# Public Policy Diagnostics Notes

Policy diagnostics should track implementation dynamics, not only policy intensity.

## Useful diagnostics

- System state trajectory
- Policy intensity trajectory
- Institutional capacity trajectory
- Trust trajectory
- Administrative burden trajectory
- Uptake trajectory
- Side-effect trajectory
- Performance gap
- Scenario comparison
- Validation checks

The key modeling question is not whether a policy exists, but whether the policy changes the system through a plausible mechanism without producing unacceptable burden, side effects, inequity, or legitimacy loss.
