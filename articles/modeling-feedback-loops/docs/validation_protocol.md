# Validation Protocol

For this synthetic companion model, validation checks workflow integrity and plausible numerical behavior.

## Checks

- Feedback trajectories are generated.
- Delayed feedback scenarios are generated.
- Target crossings are nonnegative.
- State values remain finite.
- Stock values remain nonnegative.
- Policy-resistance diagnostics are exported.
- Loop diagnostics and taxonomy are exported.
- Summary tables are generated.
