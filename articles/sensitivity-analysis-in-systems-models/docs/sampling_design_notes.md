# Sampling Design Notes

Sampling design determines how the uncertainty space is explored.

## Companion designs

- Local one-at-a-time variation
- Monte Carlo sampling
- Latin-hypercube-style stratified sampling
- Randomized compiled-language screening

## Professional extension

Applied sensitivity analysis should document:

- Parameter ranges
- Distribution choices
- Correlations among uncertain inputs
- Scenario constraints
- Number of model evaluations
- Random seeds
- Run metadata
- Output metrics
- Computational limits

Sensitivity results are only as meaningful as the uncertainty space they test.
