# Responsible Use

Sensitivity analysis can influence how model uncertainty is communicated in policy, infrastructure, sustainability, health, climate, and organizational settings.

Do not use these synthetic workflows for real decisions without:

- Justified parameter ranges
- Domain validation
- Stakeholder review
- Structural sensitivity checks
- Uncertainty communication
- Reproducible run metadata
- Clear documentation of exclusions
- Sensitivity to multiple output metrics

A responsible sensitivity analysis should make assumptions more visible and contestable. It should not create false confidence by testing only convenient ranges.
