# Model Boundary

This companion folder represents a simplified nonlinear systems model used for sensitivity-analysis demonstrations.

## Included

- Nonlinear growth
- Carrying capacity
- Extraction pressure
- Delayed recovery feedback
- External shock intensity
- Local one-at-a-time sensitivity
- Monte Carlo global sensitivity
- Latin-hypercube-style stratified sampling
- Rank-based parameter screening
- Validation diagnostics

## Excluded

- Real empirical calibration
- Real climate, infrastructure, economic, health, or ecological data
- Formal Sobol implementation
- Formal Morris trajectories
- Parameter dependence structure
- Structural model comparison beyond documented variants

## Boundary caution

The workflows are synthetic. They demonstrate sensitivity-analysis architecture and reproducible workflow design rather than empirical findings.
