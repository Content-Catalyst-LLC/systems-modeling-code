# Validation Protocol

Validation depends on model purpose.

## Basic checks

- Scripts run from a clean checkout.
- Inputs are documented.
- Outputs are reproducible.
- Scenario parameters are visible.
- State variables remain within plausible ranges.
- Model outputs are compared against target ranges.
- Sensitivity diagnostics identify influential assumptions.

## Interpretation checks

- Are conceptual claims supported by modeled behavior?
- Where does the model contradict the conceptual map?
- Are gaps used as learning signals rather than ignored?
- Are limitations documented?
