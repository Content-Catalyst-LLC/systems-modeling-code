# Responsible Use

This folder demonstrates how conceptual systems thinking can become formal systems modeling.

Use it to:

- Clarify assumptions
- Translate conceptual maps into model variables
- Compare scenarios
- Identify conceptual-formal gaps
- Support reproducible learning

Do not use it to:

- Treat synthetic outputs as empirical evidence
- Hide contested assumptions inside equations
- Claim precision beyond model validity
- Replace stakeholder knowledge
- Treat a model as a final authority
