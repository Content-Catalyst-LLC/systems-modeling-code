# Boundary Translation Notes

A systems map may include many relationships. A model must select.

## Boundary questions

- What is represented formally?
- What remains qualitative?
- What is treated as external?
- Which stakeholders are visible?
- Which harms or burdens are excluded?
- Which time horizon is used?
- What happens if the boundary expands?

## Common translation risks

- Reducing trust, legitimacy, or resilience to weak proxies
- Treating institutional assumptions as neutral parameters
- Excluding delayed or distributed harms
- Optimizing measurable outputs while ignoring public value
