# Agent-Based Adoption and Diffusion Modeling Protocol

This case models adoption as a local decision process among heterogeneous agents connected by influence networks.

## Core steps

1. Define what counts as adoption.
2. Define agent groups and attributes.
3. Generate or import a social, institutional, geographic, or influence network.
4. Seed initial adopters.
5. Define adoption pressure from benefit, peer influence, intervention, cost, trust, and resistance.
6. Update adoption states over time.
7. Track aggregate adoption and group-level adoption.
8. Compare scenarios.
9. Test sensitivity.
10. Communicate assumptions, uncertainty, and valid use.

## Core logic

Diffusion is not imposed as a smooth curve. The adoption curve emerges from heterogeneous local agent decisions.
