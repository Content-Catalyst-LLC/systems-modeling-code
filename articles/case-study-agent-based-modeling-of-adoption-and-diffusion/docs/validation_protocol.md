# Validation Protocol

Synthetic validation checks:

- Scenario runs are generated.
- Adoption shares remain between zero and one.
- Adopter counts remain nonnegative.
- Adoption gaps remain between zero and one.
- Scenario summaries are generated.
- Agent attributes remain in expected ranges.

Applied validation would require observed adoption data, empirical network information, stakeholder review, calibration, sensitivity analysis, and comparison against held-out adoption patterns.
