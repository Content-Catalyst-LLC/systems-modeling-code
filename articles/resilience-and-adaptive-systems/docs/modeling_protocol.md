# Resilience and Adaptive Systems Modeling Protocol

Core questions:

- What system and function must remain viable?
- Resilience to what disturbance or stressor?
- Resilience for whom and over what time horizon?
- What performance loss is acceptable?
- How quickly does the system recover?
- Does repeated disturbance erode or strengthen adaptive capacity?
- Which thresholds or regimes matter?
- Does adaptation preserve a viable system or entrench a harmful one?
- Does local resilience shift risk elsewhere?

The workflow compares baseline adaptation, weakened capacity, compound stress, learning investment, high redundancy, fragile efficiency, transformative adaptation, and chronic stress.
