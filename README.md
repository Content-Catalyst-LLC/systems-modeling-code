# Systems Modeling Code

Open computational companion repository for the Systems Modeling knowledge series.

This repository supports article-level code, data, documentation, and reproducible workflows for systems modeling, system dynamics, stock-and-flow models, agent-based models, network models, scenario simulations, integrated assessment concepts, digital twins, resilience modeling, policy simulation, and responsible model interpretation.

## General Principles

1. Make system boundaries explicit.
2. Document assumptions and parameter values.
3. Separate synthetic teaching data from real-world datasets.
4. Treat simulations as structured exploration, not automatic prediction.
5. Test sensitivity and uncertainty.
6. Preserve reproducible workflows.
7. Communicate limitations clearly.
8. Keep examples educational, auditable, and extensible.
